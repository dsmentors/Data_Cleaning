# Data Cleaning Learning Session

An interactive single-page web application to learn core data cleaning concepts such as duplicates, missing values, outliers, and column removal through guided explanations, decision support, comparisons, and quizzes.

## Demo

Live page URL here:        

`https://selvamani1992.github.io/Data_Cleaning/`

## Features

- Clean, interactive HTML-based learning experience.
- Covers key data cleaning topics: duplicates, missing values, outliers, and drop columns.
- Simple top navigation for moving between sections.
- Overview section to understand the data cleaning workflow.
- Decision tree style guidance to help choose the right cleaning approach.
- Comparison section to understand when to use each method.
- Quiz section for self-assessment.
- Responsive layout suitable for desktop and mobile screens.

## Learning Sections Included

### Home
Introduces the purpose of the session and highlights why data cleaning is important.

### Overview
Explains the data cleaning process and foundational concepts.

### Drop Columns
Helps identify columns that can be removed when they are irrelevant, redundant, or too sparse.

### Duplicates
Describes duplicate records and common ways to identify and handle them.

### Missing Values
Explains different types of missing data and typical treatment strategies.

### Outliers
Introduces outliers and when to keep, transform, or remove them.

### Compare
Provides a side-by-side understanding of cleaning decisions and trade-offs.

### Core Sub-sections in each Sections
There 4 main sections in each tabs
- Overview
- Decision Tree
- Compare
- Quiz
